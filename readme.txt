To start the app for development, and testing:

NVector runs on nw.js short for node webkit. It bundles web based code into a package that can be deployed acrossed multiple platforms. Make sure that is installed with npm:

npm install nwjs

be sure to be using an sdk build to include the web debugger and a current build without the sdk for deployment. 






To build the application for deployment you need to use gulp. 

with gulp installed you can simply run

gulp nw

from the root page of the application
